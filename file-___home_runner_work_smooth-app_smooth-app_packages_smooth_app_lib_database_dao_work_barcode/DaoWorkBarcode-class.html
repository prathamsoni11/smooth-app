DaoWorkBarcode

DaoWorkBarcode class Null safety

Work table that contains barcodes.

The typical use case is for bulk product downloads. The first step would be to populate this table with the barcodes you're interested in, e.g. all the local barcodes or the top 1k barcodes. The second step would be to download the products referenced in that table.
