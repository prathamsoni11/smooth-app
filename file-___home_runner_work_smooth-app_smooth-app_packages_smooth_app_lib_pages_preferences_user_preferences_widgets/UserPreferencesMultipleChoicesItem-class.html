UserPreferencesMultipleChoicesItem

UserPreferencesMultipleChoicesItem<T> class Null safety

A preference allowing to choose between a list of items. Before clicking on an item, the selected value is displayed via its title and subtitle

labels contains all the visible labels for the user in the dialog For each item, a descriptions can be provided (displayed below labels) values are of type T and are returned according to the selected value A currentValue can be provided to auto-select a value
