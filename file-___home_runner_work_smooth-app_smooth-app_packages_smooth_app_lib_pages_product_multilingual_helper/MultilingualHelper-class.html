MultilingualHelper

MultilingualHelper class Null safety

Helper for multilingual inputs (e.g. product name).

There are 2 versions of the input:

  1. the old one - only one text per product
  2. the multilingual one Typically, the old version will be used with "old" data, that have not downloaded yet the more "recent" multilingual fields (e.g. ProductField.NAME_ALL_LANGUAGES).
