AnalyticsHelper

AnalyticsHelper class Null safety

Helper for logging usage of core features and exceptions Logging:

  • Errors and Problems (sentry)
  • App start
  • Product scan
  • Product page open
  • Knowledge panel open
  • personalized ranking (without sharing the preferences)
  • search
  • external links
